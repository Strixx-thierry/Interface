# Interfaces — Unity C# Assignment

Demonstrates C# **interfaces** in Unity. An `IShape` interface declares the two shared methods (`CalculateArea`, `CalculatePerimeter`), implemented by three shapes: **Trapezium**, **Circle**, and **Nonagon**.

## Run it
1. Open the project in Unity (6.x).
2. Open `Assets/Scenes/SampleScene.unity`.
3. Open **Window → General → Console**.
4. Press Play results print to the Console via `Debug.Log`.

## Code
`Assets/Scripts/Interfaces/InterfaceTestScript.cs` — interface, shape classes, and the `Start()` test driver.
